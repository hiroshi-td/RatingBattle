## ポケモンのレーティングバトルのシミュレーション

Javaで実装しました．

### 仕様および本家と異なる前提
+ プレイヤー数(noOfPlayer)は偶数指定．
+ 全プレイヤーは一律に指定対戦数をこなす．
+ 引き分けは実装上ほぼ起きない．
+ 過度なレート差マッチング防止機構は存在しない．
+ 各プレイヤーの戦闘力は正規分布を仮定し，パラメータは一律同じ．
  + 実装次第で変更可能．

### 実行方法
